<h1 align="center">Dotfiles</h1>

## How to setup

1. Clone this repo:

    ```shell
    git clone git@github.com:pourya7/.dotfiles.git $HOME/.dotfiles
    cd $HOME/.dotfiles
    ```
2. Install Stow:

    ```shell
    # On Ubuntu
    sudo apt install stow

    # On Mac
    brew install stow
    ```
3. Run `stow NAME_OF_DIRECTORY` to setup

## Updating
To update the dotfiles pull this repo
